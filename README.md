# Study Now

A minimalist focus timer desktop app built with Tauri

## What It Does

Study Now is a simple, distraction-free focus timer designed to help you stay productive. It uses a Pomodoro-style approach with customizable sessions and automatic break management to keep you focused on your work.

## Features

  * Pomodoro-style study sessions with customizable duration (1 to 180 minutes)
  * Visual countdown timer with circular progress indicator
  * Automatic break management with configurable length (10-35% of session time)
  * Daily streak tracking to keep you motivated
  * Comprehensive session statistics including total focus time, total sessions, and longest streak
  * Clean, dark-themed interface with custom titlebar
  * Completion sound to celebrate your progress
  * Session validation to ensure you complete at least 90% of your session for it to count
  * Optional focus topic labeling for each session

## Tech Stack

  * Frontend: Vanilla JavaScript, HTML5, CSS3
  * Desktop: Tauri 2.10.1 with Rust backend
  * Build: Tauri CLI with NSIS installer
  * Storage: LocalStorage for persistence
  * UI: Custom CSS with smooth transitions and micro-interactions

## Highlights

  * Lightweight, fast native performance
  * Minimal system resource usage
  * Focused user experience with motivational microcopy
  * Dark theme by default
  * Custom titlebar with window controls
